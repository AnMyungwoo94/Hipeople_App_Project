## ▶ 앱 소개
 "간단한 인삿말로 인연을 만들어보세요 “

 -원하는 이상형 선택가능
 
 -1:1 채팅 가능
 
 -마음에 드는 이성 좋아요 선택
 
 -내가 채팅한 사람 목록 보이기
 
 -구글맵으로 주변 맛집, 카페 검색하기
 
 -익명 게시판 기능으로 자유롭게 소통가능
 
 -회원가입 및 자동로그인 기능
